CSS Specificity

Inline Style > ID Selector > Class/Attribute/Pseudo-class > Element Selector > Universal Selector

Attribute Selectors: [type="text"]

  • Inline Styles: style="color: orange;"
  • ID Selectors: #idselector
  • Class Selectors: .yellow, .red, .purple
  • Element Selectors: 'h1', 'p', 'div'
  • Universal Selector: '*'
